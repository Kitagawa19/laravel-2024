# 概要
履修登録サイト

## 使用言語・FW
PHP,Laravel,MySQL,TailwindCSS

## Step

1.リポジトリのクローン  
-> ターミナルを開き、以下のコマンドを入力してリモートリポジトリの内容をローカルにクローンします：  
```
git clone git@github.com:Kitagawa19/laravel-2024.git
cd laravel-2024                                                
```
2.sailのインストール
```
sail composer require laravel/breeze --dev
sail artisan breeze:install                                               
```
3.サーバーの起動
```
sail up                                             
```
